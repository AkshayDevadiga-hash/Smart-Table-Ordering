## 1. ❌ Review Not Showing on Admin Page

### Problem:

* Reviews are submitted but not visible on admin page

### Fix Steps:

#### Backend:

* Verify `GET /api/reviews` returns:

  * `order_id`
  * `rating`
  * `comment`
  * `created_at`
* Join with orders table to include:

  * `table_id`

Example:

```
SELECT r.*, o.table_id
FROM reviews r
JOIN orders o ON r.order_id = o.id;
```

## 3. 💰 Table-wise Revenue Report

### Requirement:

* Show revenue generated per table

---

### Backend:

Create query:

```
SELECT table_id,
       COUNT(id) as total_orders,
       SUM(total) as total_revenue
FROM orders
WHERE payment_status = 'paid'
GROUP BY table_id;
```

---

### API:

* Add endpoint:

  * `GET /api/reports/table-revenue`

Response example:

```
[
  {
    table_id: 1,
    total_orders: 5,
    total_revenue: 2500
  }
]
```

---
